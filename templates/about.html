This is an Unofficial Habits app based off the best selling book "Atomic Habits"

If you don't have it I recommend clicking the image below,to first get the book

as I don't explain any of the concepts used in this app.

So if you've already read the book, you'll want to go straight to "Create a Habit" or "Destroy a Habit".

After you've filled out the questionaire, you're new habit will appear in the "Habit List".

Voila! you'll then have a habit sheet, with the option to print it out, make changes or review it on the App.
